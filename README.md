# Expense Tracker (Трекер расходов)

**Автор:** Гирштейн Кирилл Игоревич

## Описание
GUI-приложение для отслеживания личных расходов с фильтрацией по категориям и датам, подсчетом сумм за период и сохранением данных в JSON.

## Возможности
- Добавление расходов (сумма, категория, дата)
- Фильтрация по категории и периоду
- Подсчет общей суммы за выбранный период
- Сохранение/загрузка данных в JSON
- Валидация ввода

## Установка и запуск
1. Установите Python 3.8+
2. Сохраните файл `expense_tracker.py`
3. Запустите: `python expense_tracker.py`

## Использование
1. Введите сумму (положительное число)
2. Выберите категорию из списка
3. Укажите дату в формате YYYY-MM-DD
4. Нажмите "Добавить расход"
5. Используйте фильтры для анализа данных

## Примеры
- Добавьте расход: 500 руб, "еда", 2026-04-28
- Отфильтруйте по категории "еда"
- Посмотрите итоговую сумму за период
